   ffmpeg 更新
   - (johnvansickle 静态更新)[https://www.johnvansickle.com/ffmpeg/]
   - (BtbN-build)[https://github.com/BtbN/FFmpeg-Builds]
   - (官网库)[https://ffmpeg.org/download.html]
  
3. 注册systemd， 参考(rtsp-server.service)
4. 启动推流脚本 ./start.sh
